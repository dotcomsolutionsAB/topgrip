# Images Directory

This directory contains all images for the Topgrip Industries website.

## Directory Structure

- `logo/` - Company logo and branding assets
- `team/` - Team member photos
  - `zakir-hussain.jpg` - CEO
  - `abdul-kader-kagalwala.jpg` - Head of Operations
  - `yusuf-habib.jpg` - Engineering
  - `mufaddal-zakirhusain.jpg` - Manufacturing & Innovation
- `solutions/` - Solution category images
  - `automotive.jpg` - Automotive solutions image
  - `railways.jpg` - Railways solutions image
  - `oil-gas.jpg` - Oil & Gas solutions image
- `general/` - General website imagery and backgrounds

## Image Requirements

- **Format**: JPG, PNG, or WebP
- **Optimization**: Compress images for web use
- **Team Photos**: 400x400px minimum, square format recommended
- **Solution Images**: 1200x600px recommended
- **Logo**: SVG or high-resolution PNG

## Notes

- The website includes fallback placeholders for missing images
- All images should be optimized for web to ensure fast loading
- Use descriptive filenames (lowercase, hyphens for spaces)
- Provide alt text for all images (already included in HTML)
